# Q2: Gateway 启动失败？

💡 提示：如果 3000 端口被占用，可以修改配置文件使用其他端口。

# Q3: API Key 无效？

  • 检查 API Key 是否正确复制（无多余空格）
  • 确认账号已实名认证
  • 检查账号余额是否充足

⚠️ 注意：API Key 区分大小写，复制时请仔细核对。

# Q4: Chrome 扩展连接失败？

  • 确认 Chrome 扩展已安装
  • 确认扩展中 Gateway URL 配置正确
  • 重启 Gateway 服务

💡 提示：确保 Chrome 扩展和 Gateway 版本兼容。
