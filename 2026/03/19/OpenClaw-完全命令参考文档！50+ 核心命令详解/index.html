📋 命令选项详解

openclaw status

显示频道健康和最近会话状态

选项：

  • --all - 完整诊断
  • --deep - 深度探测频道
  • --json - JSON 输出
  • --usage - 显示模型使用情况
  • --timeout <ms> - 探测超时

openclaw gateway

运行、检查和查询 WebSocket Gateway

常用选项：

  • --port <port> - 端口号
  • --bind <mode> - 绑定模式
  • --auth <mode> - 认证模式
  • --token <token> - 共享令牌
  • --force - 强制重启
  • --verbose - 详细日志

openclaw browser

管理 OpenClaw 专用浏览器

常用子命令：

  • start - 启动浏览器
  • open - 打开 URL
  • snapshot - 捕获页面快照
  • screenshot - 截图
  • click - 点击元素
  • type / press - 输入文本/按键
  • fill - 填写表单
  • pdf - 保存为 PDF
